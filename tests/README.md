# Tests Directory

This directory contains test files for FastBB.

## Running Tests

```bash
# Run backend tests
cd src/backend
npm test

# Run frontend tests
cd src/frontend
npm test
```

## Test Structure

```
tests/
├── backend/      # Backend unit and integration tests
├── frontend/     # Frontend component tests
└── fixtures/     # Test data fixtures
```

## Writing Tests

Use Jest for backend tests and Vitest for frontend tests.
